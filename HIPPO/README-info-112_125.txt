HIPPO Combined Discrete Flask and GC Sample GHG, Halocarbon, and Hydrocarbon
Data
https://data.eol.ucar.edu/dataset/112.125

Summary
This data set contains the results of analyses of discrete samples and the
analyses of flask samples for all Missions, 1 through 5, of the HIAPER
Pole-to-Pole Observations (HIPPO) study of carbon cycle and greenhouse gases.
The Missions took place from 08 January 2009 to 08 September 2011. The data
are provided in one space-delimited ASCII format file. Please refer to the
readme for more information. Note that EOL Version 1.0 corresponds to R.
20121129 previously served by ORNL.
